## Description

This is a simple **Python program** that simulates ordering ice cream in a shop called **“Dulce Sabor.”**

The user chooses how many ice creams they want (maximum of 5) and then selects the flavor for each one.

## Available Flavors

* Vanilla
* Chocolate
* Strawberry

## How It Works

1. The program shows a welcome message.
2. The user enters the number of ice creams (maximum 5).
3. The user chooses the flavor for each ice cream.
4. The program counts how many of each flavor were selected.
5. At the end, it shows the final results.

## Result

The program displays how many **vanilla, chocolate, and strawberry** ice creams were ordered.
